# URPOcean

Require Unity 2020.3+ (URP 10.x+) or 2021.3+ (URP 12.x+) 

Suitable for wide-range mobile devices, hardware features need #pragma target 3.0 or higher.  
Now support for gles 3.0 and gles2.0 ext(Experimental).
